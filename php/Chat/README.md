This demo is a PHP client for the [ZeroC Chat Demo](https://zeroc.com/chat/index.html). This demo connects to a single chat room, allowing you to chat with other chat room participants.

By default, the demo connects throught a WebSocket proxy hosted on zeroc.com to a chat server hosted on demo.zeroc.com.

To run this demo configure your PHP-enabled web server to serve the directoy where this file exists.
